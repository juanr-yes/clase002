Creando un repositorio en Github

  1. Crear el repositorio en github
  2. Crear la carpeta donde se almacenara nuestro proyecto
  3. Abrir la CARPETA (no un archivo) con Visual Studio Code
  4. Abrir la terminal integrada
  5. git init este comando inicializa el repositorio
  6. git remote add origin https://github.com/nombredeusuario/repo.git esta instruccion agrega el origen
  7. git add nombredearchivo esta instruccion agrega nuestros cambios a stage, o nuestra piscina de cambios
  8. git commit -m "nombre del commit" agrega un nombre a nuestra piscina de cambios
  9. git push -u origin master crea rama dentro del repositorio y subir los cambios
  10. Finalmente, cuando seguimos haciendo cambios, ejecutamos: git push (Luego de haber agregado los cambios y realizado commit)
